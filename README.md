# Bit: Library for Bit implementation in C++

## Definitions
 - Bit a(b): a=b%2 (0 or 1)
 - Bit a=b: a=b&2 (0 or 1)

## Functions
 - GetBit(): Get Bit
 - GetFlipBit(): Get Flipped Bit

## Operations
 - Addition: b1+b2, b1+=b2
 - Subtraction: b1-b2, b1-=b2
 - Multiplication: b1*b2, b1*=b2
 - Subtraction: b1/b2, b1/=b2
 - IsEqual: b1==b2
 - IsNotEqual: b1!=b2
